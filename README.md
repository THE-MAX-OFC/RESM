# Useage
First time process
```
apt update
apt upgrade -y
apt install git
pkg install python
pkg install python2
pip install requests
pip install rich
pip install bs4
git clone https://github.com/THE-MAX-OFC/RESM.git
cd RESM
python run.py
```
After installation
```
cd RESM
python run.py
```
